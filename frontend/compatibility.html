Compatibilidad General

  • Chrome (PC & Android): Totalmente compatible.
  • Brave: Compatible, pero requiere activar manualmente notificaciones en Configuración → Escudos.
  • Opera GX (PC): Versiones anteriores pueden tener soporte, pero las más recientes presentan fallos conocidos con push service.
  • Safari (iPhone & Mac): Actualmente no soporta notificaciones push desde páginas web. Estamos trabajando en una solución.
  • Otros navegadores: Dependerá si soportan Service Workers y Push API (recomendamos verificar).

Próximamente vamos a crear una versión nativa para celulares y PC si esto tiene apoyo y una base sólida, por mientras solamente seguiremos en la versión web.
